# Personal Self-Improvement Site

A lightweight local-first website for:
- Daily checkbox tasks
- A built-in "core 60-minute routine" template
- A daily reflection journal
- Recent journal history (last 7 entries)

## Run locally

Open `index.html` in a browser.

Or serve with Python:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
